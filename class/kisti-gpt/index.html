당신은 파이썬 전문가입니다.
        저는 파이썬 초보자이며 내 데스크탑의 VS코드에서 실행할 코드를 만들고 싶습니다.
        
        [요구사항] 
        1) 필요한 라이브러리가 있다면 코드 실행 시 자동으로 설치되도록 해주세요.
           (try-except 구조를 사용하여 없는 경우에만 설치하고, sys.executable을 활용해주세요)
        
        2) 재사용이 가능하도록 주요 기능은 함수로 구성해주세요.  
           각 함수에는 한국어 docstring을 작성해주세요.
        
        3) 주석은 단계 번호 형식으로 달아주세요.  
           예: # (1) CSV 파일 불러오기  
           중요한 줄/블록마다 **무엇을 하는지 + (이 작업을 하는 이유)**를 설명해주세요.
        
        4) 코드 흐름을 이해하기 쉽게 main() 함수와  
           `if __name__ == "__main__": main()` 구조를 포함해 주세요.
        
        5) 각 주요 단계마다 **[DEBUG]**로 시작하는 디버그용 print를 넣어주세요.  
           (예: 진행 단계, 변수 크기/타입, 샘플 값 등)
        
        6) 코드 작성 후 아래 내용을 추가로 작성해주세요.  
           - 코드 전체 동작 요약
           - 초보자가 자주 실수하는 부분 3개
        
        7) 한국어 데이터 처리 시 반드시 다음을 포함해주세요:
           - 그래프에 한글 포함 시: '맑은 고딕'(Malgun Gothic) 폰트로 설정하고,
             마이너스 기호(-) 깨짐 방지 코드도 포함
